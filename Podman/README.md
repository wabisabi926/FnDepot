<p>🐭使用Podman管理容器、Pod和镜像，为飞牛os无痛添加另一个容器环境，不修改系统环境、目录权限。启动后以systemd服务后台运行，支持rootful有根模式与rootless无根模式同时运行与切换。</p><p>⭐使用说明：</p><p>本项目不含容器管理面板，需搭配Dpanel等面板使用，双模式的容器数据、镜像、配置文件等各自独立。</p><p>受限于飞牛安装环境，本项目root模式部署，无根模式为启动后切换"UID:1000"用户激活用户systemd服务，受限"/var/tmp"目录权限，无根模式不支持在终端直接拉取镜像(面板拉取镜像不影响)，需修改当前会话"TMPDIR"至用户可读写路径。</p><p>Dpanel面板添加sock监听：系统>多服务端>添加服务端>"服务类型podman;连接方式sock">Docker API地址：rootful有根模式"unix:///run/podman/podman.sock";rootless无根模式"unix:///run/user/1000/podman/podman.sock"</p><p>💡故障说明：</p><p>切换运行模式、停用Podman之前建议先停止所有容器，卸载Podman前建议先停止并删除所有容器、镜像和挂载卷。如遇安装时初始化应用目录结构失败、卸载残留容器挂载点，可使用umount指令手动卸载挂载点或重启飞牛os后删除残留文件；如遇无根模式启动失败，可重启飞牛os后再试。</p>
